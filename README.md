[Acesse Fruit Run!](https://fruit-run.netlify.app/)

Projeto de desenvolvimento de um jogo de computador simples, criado como parte da disciplina de Jogos Digitais no Ensino Médio. Desenvolvido utilizando o software Construct 3, com foco na implementação de lógica básica de eventos, sistema de pontuação e mecânica de controle de personagem.

Comandos: Para pular: clique setinha para cima
          Para reiniciar a partida: clique R
